Blow out es un endless runner que surge como projecto de fin de grado, formado por 3 miembros del centro, del cual fui partcipe y ayude en la creacion del codigo y animaciones entre otras funciones.
Para abrir correctamente el projecto hace falta descargar el zip, el cual estara adjuntado al final, el motor para desarrollar el videojuego fue unreal engine.
